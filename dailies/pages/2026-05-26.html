DVAO: Dynamic Variance-adaptive Advantage Optimization for Multi-reward Reinforcement Learning

Published: 2026-05-24

Link: http://arxiv.org/pdf/2605.25604

1. 📘 Topic and Domain: The paper presents a reinforcement learning method for multi-objective alignment of large language models within the GRPO framework.
2. 💡 Previous Research and New Ideas: It extends GRPO and existing static scalarization methods (Reward/Advantage Combination) by introducing dynamic variance-adaptive weighting to balance multiple objectives.
3. ❓ Problem: It solves the training instability and suboptimal trade-offs caused by Reward Combination's exploding advantage magnitudes and Advantage Combination's static, isolated normalization.
4. 🛠️ Methods: The proposed DVAO method dynamically adjusts objective weights based on per-objective reward variance within rollout groups and mathematically proves bounded advantages and cross-objective regularization.
5. 📊 Results and Evaluation: On mathematical reasoning and tool-use benchmarks with Qwen models, DVAO consistently dominates the Pareto frontier, outperforming baselines in both accuracy and compliance metrics with stable training.
1. 📘 主题与领域: 该论文提出了一种在GRPO框架下用于大语言模型多目标对齐的强化学习方法。
2. 💡 先前研究与新思路: 它通过引入动态方差自适应加权，扩展了GRPO及现有的静态标量化方法（奖励/优势组合）以平衡多个目标。
3. ❓ 问题: 它解决了奖励组合导致的优势幅度爆炸以及优势组合静态孤立归一化所引起的训练不稳定性与次优权衡问题。
4. 🛠️ 方法: 所提出的DVAO方法基于rollout组内各目标的奖励方差动态调整目标权重，并在数学上证明了其优势幅度有界及跨目标正则化特性。
5. 📊 结果与评估: 在基于Qwen模型的数学推理和工具使用基准测试中，DVAO在准确率和合规性指标上均优于基线方法，以稳定的训练表现持续主导帕累托前沿。

SkillOpt: Executive Strategy for Self-Evolving Agent Skills

Published: 2026-05-21

Link: http://arxiv.org/pdf/2605.23904

1. 📘 Topic and Domain: The paper introduces SkillOpt, a text-space optimizer that treats reusable natural-language skill documents as the external trainable state for frozen LLM agents, operating in the domain of AI agent adaptation and procedural skill optimization. 2. **💡 Previous Research and New Ideas:** It builds on prior work in prompt optimization, skill evolution, and trajectory-driven reflection, but innovates by applying deep-learning-style optimization controls—such as bounded edit budgets, held-out validation gates, rejected-edit buffers, and epoch-wise slow/meta updates—to the skill document itself. 3. **❓ Problem:** It addresses the lack of a systematic, reproducible training procedure for agent skills, as existing methods rely on hand-crafting, one-shot generation, or loosely controlled self-revision that fail to reliably improve performance under feedback. 4. **🛠️ Methods:** SkillOpt uses a separate optimizer model to analyze rollout trajectories, propose structured add/delete/replace edits, rank and clip them under a textual learning-rate budget, and accept updates only when they strictly improve a held-out validation score, while retaining rejected edits as negative feedback and consolidating long-horizon lessons via epoch-wise slow/meta updates. 5. **📊 Results and Evaluation:** Across six benchmarks, seven target models, and three execution harnesses, SkillOpt is best or tied on all 52 evaluated cells, lifting GPT-5.5 accuracy by over 23 points on average while producing compact, transferable skill artifacts.

Macaron-A2UI: A Model for Generative UI in Personal Agents

Published: 2026-05-23

Link: http://arxiv.org/pdf/2605.24830

1. 📘 Topic and Domain: The paper introduces Macaron-A2UI, a learning-based approach for Generative UI in personal agents that dynamically synthesizes lightweight, executable user interfaces alongside natural language to move beyond static plain-text chat in human-computer interaction. 2. **💡 Previous Research and New Ideas:** Building on prior work in conversational AI, code generation, and existing GUI agents, the paper newly frames agent-side UI generation as a unified learning problem under the declarative A2UI protocol, demonstrating that models can internalize executable interface generation without relying on long schema prompts at inference time. 3. **❓ Problem:** It addresses the bottleneck of static plain-text interaction for complex user-centric tasks and the field's lack of large-scale UI-grounded dialogue supervision, controlled evaluation benchmarks separating protocol validity from interaction quality, and evidence that models can generate compliant, useful UI without heavy explicit schema guidance. 4. **🛠️ Methods:** The authors construct a 14,245-sample A2UI corpus from heterogeneous dialogues via a hybrid rule-and-LLM annotation pipeline with deterministic validation, introduce the A2UI-Bench benchmark with atomic, depth, and width tasks evaluated across protocol validity (L1), task construction quality (L2), and user experience (L3), and train 30B to 754B models using a parameter-efficient two-stage recipe of LoRA-based supervised fine-tuning followed by reward-driven GRPO reinforcement learning. 5. **📊 Results and Evaluation:** The best Macaron-A2UI model (Venti, 754B) achieves an overall score of 75.6 on A2UI-Bench without explicit schema hints, surpassing the strongest full-schema frontier baseline and confirming that generative UI competence can be successfully learned and internalized.
