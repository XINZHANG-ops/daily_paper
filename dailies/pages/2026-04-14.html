Paper 1

OmniShow: Unifying Multimodal Conditions for Human-Object Interaction Video Generation

Published: 2026-04-12

Link: http://arxiv.org/pdf/2604.11804

1. 📘 Topic and Domain: The paper addresses Human-Object Interaction Video Generation (HOIVG), which involves synthesizing high-quality videos conditioned on multiple modalities including text, reference images, audio, and pose, with applications in e-commerce, short video production, and interactive entertainment.
2. 💡 Previous Research and New Ideas: Built upon Waver 1.0 (a 12B MMDiT-based latent diffusion model), the paper introduces three novel contributions: Unified Channel-wise Conditioning for efficient multimodal injection, Gated Local-Context Attention for precise audio-visual synchronization, and a Decoupled-Then-Joint Training strategy to leverage heterogeneous sub-task datasets through model merging.
3. ❓ Problem: The paper solves the challenge of unifying four distinct multimodal conditions (text, reference images, audio, and pose) in an end-to-end framework, addressing the trade-off between controllability and generation quality, overcoming data scarcity through heterogeneous dataset utilization, and filling the evaluation gap with a dedicated benchmark (HOIVG-Bench).
4. 🛠️ Methods: The method employs Unified Channel-wise Conditioning using pseudo-frame tokens and channel concatenation for seamless injection, Gated Local-Context Attention with audio context packing and masked cross-attention for temporal alignment, and a multi-stage Decoupled-Then-Joint Training process that first trains specialized R2V and A2V models separately before merging them for joint fine-tuning with pose conditions.
5. 📊 Results and Evaluation: OmniShow achieves state-of-the-art or highly competitive performance across R2V, RA2V, and RP2V settings, being the smallest 10B-scale model (12.3B parameters) while demonstrating superior reference consistency, audio-visual synchronization, and video quality metrics on the HOIVG-Bench benchmark with 135 curated samples.
1. 📘 主题与领域: 本文研究人-物交互视频生成（HOIVG），涉及基于文本、参考图像、音频和姿态等多种条件合成高质量视频，可应用于电子商务演示、短视频制作和交互娱乐等领域。
2. 💡 先前研究与新思路: 基于Waver 1.0（12B参数的MMDiT潜扩散模型），论文提出三项创新：统一通道级调节实现高效多模态注入、门控局部上下文注意力实现精确音视频同步，以及解耦-联合训练策略通过模型融合利用异构子任务数据集。
3. ❓ 问题: 论文解决了在端到端框架中统一四种不同多模态条件（文本、参考图像、音频和姿态）的挑战，同时兼顾可控性与生成质量的平衡，通过异构数据集利用克服数据稀缺问题，并建立专用基准HOIVG-Bench填补评估空白。
4. 🛠️ 方法: 采用伪帧令牌和通道拼接的统一通道级调节实现无缝注入；使用音频上下文打包和掩码交叉注意力的门控局部上下文注意力实现时间对齐；通过多阶段解耦-联合训练流程，先分别训练专门的R2V和A2V模型，再通过模型融合进行联合微调并引入姿态条件。
5. 📊 结果与评估: OmniShow在R2V、RA2V和RP2V设置中均达到最优或极具竞争力的性能，作为最小的10B级模型（12.3B参数），在HOIVG-Bench基准（含135个精选样本）上展现出卓越的参考一致性、音视频同步性和视频质量指标。

Paper 2

Pseudo-Unification: Entropy Probing Reveals Divergent Information Patterns in Unified Multimodal Models

Published: 2026-04-12

Link: http://arxiv.org/pdf/2604.10949

1. 📘 Topic and Domain: The paper focuses on Unified Multimodal Models (UMMs) that aim to synergize large language models (LLMs) with vision models for both text generation and image synthesis, investigating a phenomenon called "pseudo-unification."
2. 💡 Previous Research and New Ideas: Previous research on probing UMMs lacks model-internal insight or ignores prompt-response dependencies; this paper proposes an information-theoretic framework using matrix-based Rényi entropy in Reproducing Kernel Hilbert Spaces (RKHS) to jointly analyze how UMMs encode inputs and generate outputs across modalities.
3. ❓ Problem: The paper addresses the problem of pseudo-unification in UMMs—despite shared parameters, models exhibit divergent response behaviors where text generation shows high-entropy creativity while image synthesis enforces low-entropy fidelity, stemming from modality-asymmetric encoding and pattern-split responses.
4. 🛠️ Methods: The paper uses a two-level entropy-probing framework: (i) Prompt-level analysis measuring embedding entropy and layer-wise entropy to reveal modality-specific encoding patterns; (ii) Response-level analysis computing conditional entropy proxy to diagnose response behavior divergence across 10 representative UMMs.
5. 📊 Results and Evaluation: The analysis across ten UMMs reveals that pseudo-unification arises from dual divergence in encoding and response patterns; only Harmon (using contextual prediction for both modalities) achieves genuine unification, demonstrating that real multimodal synergy requires consistent information flow, not just shared parameters.
1. 📘 主题与领域: 该论文聚焦于统一多模态模型（UMMs），旨在将大型语言模型（LLMs）与视觉模型协同以实现文本生成和图像合成，研究了一种名为“伪统一”的现象。
2. 💡 先前研究与新思路: 先前对UMMs的探测方法缺乏模型内部洞察或忽略提示-响应依赖性；该论文提出一种信息论框架，在再生核希尔伯特空间（RKHS）中使用矩阵-based Rényi熵来分析UMMs如何跨模态编码输入和生成输出。
3. ❓ 问题: 该论文解决了UMMs中的伪统一问题——尽管共享参数，模型展现出响应行为分歧，文本生成表现为高熵创造性，而图像合成则强制低熵保真度，源于模态不对称编码和模式分裂响应。
4. 🛠️ 方法: 该论文采用两级熵探测框架：（i）提示级分析，通过测量嵌入熵和逐层熵来揭示模态特异性编码模式；（ii）响应级分析，计算条件熵代理以诊断跨10个代表性UMMs的响应行为分歧。
5. 📊 结果与评估: 对十个UMMs的分析表明，伪统一源于编码和响应模式的双重分歧；仅Harmon（对两种模态使用上下文预测）实现了真正的统一，表明真正的多模态协同需要信息流的一致性，而不仅仅是共享参数。 ===

Paper 3

CodeTracer: Towards Traceable Agent States

Published: 2026-04-12

Link: http://arxiv.org/pdf/2604.11641

1. 📘 Topic and Domain: This paper addresses the debugging and failure localization of large language model (LLM)-powered code agents that autonomously interact with software repositories and development environments.
2. 💡 Previous Research and New Ideas: The paper builds upon existing code agent frameworks (SWE-Agent, OpenHands, Terminus 2, MiniSWE-Agent) and benchmarks (SWE-bench, TerminalBench), introducing CodeTracer as a novel tracing architecture that converts heterogeneous agent run directories into structured hierarchical traces for automated failure onset localization.
3. ❓ Problem: The paper solves the problem of debugging complex code agent executions, where early missteps can trap agents in unproductive loops or cascade into hidden error chains, making it difficult to identify where and why an agent goes off track.
4. 🛠️ Methods: The paper proposes CodeTracer, which uses a three-stage pipeline: (1) evolving extraction that parses heterogeneous run artifacts through adaptive parsers, (2) tree indexing that reconstructs trajectories as hierarchical trace trees with persistent memory, and (3) diagnosis that localizes failure origins through evidence retrieval and root-cause tracing.
5. 📊 Results and Evaluation: CodeTracer achieves up to 48% macro F1 in step-level failure localization while reducing token costs, and reflective replay using its diagnostic signals consistently recovers originally failed runs under matched budgets; additionally, the companion benchmark CodeTraceBench contains 3,326 annotated trajectories spanning five backbones and four agent frameworks.
1. 📘 主题与领域: 本文聚焦于大型语言模型驱动的代码智能体（code agents）的调试与故障定位领域，这些智能体能够自主与软件仓库和开发环境进行交互。
2. 💡 先前研究与新思路: 本文基于现有的代码智能体框架（如SWE-Agent、OpenHands、Terminus 2等）和基准测试（如SWE-bench、TerminalBench），创新性地提出CodeTracer——一种将异构智能体运行目录转换为结构化层次轨迹并进行自动化故障溯源的追踪架构。
3. ❓ 问题: 本文解决的问题是如何调试复杂的代码智能体执行过程，在这些过程中，早期的小错误可能导致智能体陷入无效循环或级联形成隐藏的错误链，使得难以确定智能体在何时以及为何偏离正确轨道。
4. 🛠️ 方法: 本文提出CodeTracer，采用三阶段流程：（1）进化式抽取，通过自适应解析器解析异构运行产物；（2）树索引，将轨迹重构为具有持久记忆的层次化轨迹树；（3）诊断，通过证据检索和根因追踪定位故障起源。
5. 📊 结果与评估: CodeTracer在步骤级故障定位中达到最高48%的宏平均F1分数，同时降低令牌消耗；利用其诊断信号进行反思式重放，能够在相同预算下持续恢复原本失败的运行；配套基准测试CodeTraceBench包含3326条标注轨迹，涵盖五种模型骨干和四种智能体框架。
