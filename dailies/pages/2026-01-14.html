Published: 2026-01-12

Link: http://arxiv.org/pdf/2601.08079

1. üìò Topic and Domain: The paper focuses on executive memory for complex reasoning in tool-augmented AI agent frameworks, specifically addressing memory management and context control in language models.
2. üí° Previous Research and New Ideas: Based on previous research in agent memory and context management, it proposes a novel "executive memory" paradigm that actively manages reasoning trajectories rather than passively storing information.
3. ‚ùì Problem: The paper aims to solve the problem of reasoning traces and tool artifacts accumulating and straining the bounded working context of large language models during complex reasoning tasks.
4. üõ†Ô∏è Methods: The paper introduces MemoBrain, a copilot-style memory system that constructs dependency-aware memory over reasoning steps and manages working context through folding completed sub-trajectories and selectively flushing low-utility memory elements.
