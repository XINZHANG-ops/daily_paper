## Summary

HY-Embodied-0.5 is a family of foundation models designed specifically for real-world embodied agents, developed by Tencent's Robotics X and HY Vision teams. The system addresses the critical gap between general Vision-Language Models (VLMs) and the demanding requirements of embodied AI, which requires fine-grained visual perception for physical grounding alongside advanced reasoning capabilities for prediction, interaction, and planning in dynamic environments.

The architecture introduces several key innovations: a Mixture-of-Transformers (MoT) architecture that enables modality-adaptive computing by using non-shared parameters for visual and text tokens, effectively doubling the visual parameter capacity without introducing significant overhead; an efficient native-resolution Vision Transformer (HY-ViT 2.0) that supports arbitrary-resolution inputs through knowledge distillation from a larger internal model; and visual latent tokens that bridge vision and language modalities by appending learnable tokens to each visual element, supervised by global features from a large ViT.

The training pipeline is equally innovative, employing a three-stage approach: large-scale pre-training on 600B+ tokens covering visual perception, spatial reasoning, and embodied understanding; an iterative self-evolving post-training paradigm combining reinforcement learning with rejection sampling fine-tuning; and large-to-small on-policy distillation to transfer advanced capabilities from the 32B model to the efficient 2B variant. The 2B model achieves state-of-the-art on 16/22 benchmarks, while the 32B MoE variant achieves 67.0% average, surpassing Gemini 3.0 Pro at 63.6%.

## Key Contributions

- **Mixture-of-Transformers Architecture**: Introduces non-shared QKV and FFN parameters for visual and text tokens, enabling modality-adaptive computing that significantly improves visual modeling capacity while preventing language capability degradation from heavy visual training.

- **HY-ViT 2.0 Native-Resolution Visual Encoder**: A 400M-parameter ViT that inherently supports arbitrary-resolution inputs, trained via distillation from a larger internal model for efficient yet accurate visual representations with discrete visual reconstruction supervision.

- **Iterative Self-Evolving Post-Training**: Combines cold start data with iterative reinforcement learning (GRPO) and rejection sampling SFT to progressively improve model's thinking capabilities through an evolving training paradigm.

- **Large-to-Small On-Policy Distillation**: Transfers advanced capabilities from the 32B MoE model to the 2B variant through policy-based distillation, maximizing performance potential of compact edge models.

- **Comprehensive Robot Control Integration**: Demonstrates practical effectiveness by training a Vision-Language-Action (VLA) model achieving 85%, 80%, and 75% success rates on real-world packing, stacking, and hanging tasks respectively.

## Method

The model architecture comprises three primary components. First, the HY-ViT 2.0 visual encoder projects visual inputs into language embedding space using a 400M-parameter model trained through knowledge distillation, supporting arbitrary resolution inputs and incorporating discrete visual representation supervision with a codebook size of 2k that compresses every 8x8 patch into a single discrete code. Second, the Mixture-of-Transformers backbone employs modality-specific QKV and FFN layers alongside distinct attention mechanisms—bidirectional attention for visual tokens and causal attention for text tokens—with over half of tokens in multimodal data being vision tokens, enabling a visual next-code prediction task for enhanced visual supervision. Third, visual latent tokens are appended to each visual input sequence and supervised by global features from a large ViT, effectively bridging visual and textual content.

The pre-training corpus exceeds 600B tokens, organized into four tiers: visual perception data (62M detection samples, 36M depth estimation samples, 5M segmentation samples, 11M pointing/counting samples); embodied-centric data covering grounding, affordance, trajectory prediction, understanding, planning, and reasoning across robotics manipulation sequences; spatial-centric data including correspondence, geometry, configuration, measurement, and dynamics reasoning from ScanNet, ScanNet++, and ARKitScenes; and general understanding data. The mid-training stage leverages 12M high-quality QA pairs for complex real-world execution.

Post-training employs supervised fine-tuning on robot trajectory data and reasoning tasks, followed by reinforcement learning using GRPO (Group Relative Policy Optimization) that computes group-relative advantages by normalizing rewards within each sampled group, combined with hybrid reward designs. The iterative self-evolution process alternates between RL and rejection sampling SFT to progressively improve model capabilities. Finally, on-policy distillation transfers knowledge from MoE-A32B to MoT-2B through policy-based training.

## Results

**Benchmark Performance (MoT-2B)**:
- Achieves state-of-the-art on 16/22 benchmarks among compared generalist and specialist embodied VLMs
- Average score of 58.0% across 22 benchmarks
- Outperforms Qwen3-VL-4B by 10.2% (relative improvement)
- Outperforms RoboBrain2.5-4B by 8.6% (relative improvement)
- Comparable to Qwen3.5 on general VLM understanding tasks

**Benchmark Performance (MoE-A32B)**:
- Achieves 67.0% average across 22 benchmarks
- Surpasses Gemini 3.0 Pro (63.6%) by 3.4 absolute percentage points

**Robot Control Experiments**:
- Packing task: 85% success rate (vs. baseline significantly lower)
- Stacking task: 80% success rate (vs. baseline significantly lower)
- Hanging task: 75% success rate (significantly outperforming π0 and π0.5 baselines)

**Spatial Reasoning Benchmarks**:
- ERQA: 80.7%
- EmbSpatial-Bench: 47.3%
- VSIBench: 55.2%
- SITE-Bench: 62.7%

## Limitations & Future Work

The paper acknowledges that while HY-Embodied-0.5 demonstrates strong performance across benchmarks and robot control tasks, there remain challenges in scaling the approach to even more complex embodied scenarios and edge deployment optimization. The authors note that the iterative self-evolution paradigm requires careful balancing between reinforcement learning and supervised fine-tuning to avoid capability regression. Future work will focus on extending the framework to handle more diverse robot morphologies and integration with physical world simulations for scalable training.
