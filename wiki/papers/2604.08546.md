# When Numbers Speak: Aligning Textual Numerals and Visual Instances in Text-to-Video Diffusion Models

**Published**: 2026-04-08 | **Added**: 2026-04-10 | **arXiv**: [2604.08546](http://arxiv.org/pdf/2604.08546)

## Summary

NUMINA addresses the problem of incorrect object counts in text-to-video generation—current T2V models often generate videos with wrong number of objects because numeral tokens have weak semantic grounding and instance representations are ambiguous in latent space. The solution is a training-free "identify-then-guide" framework that exploits discriminative attention heads to extract and refine instance layouts for accurate counting.

The approach selects instance-discriminative self- and cross-attention heads, builds countable layouts via clustering, conservatively adds/removes instances using heuristic costs, and guides regeneration by modulating cross-attention.

## Key Contributions

- **Training-free approach**: No fine-tuning required, works with existing T2V models
- **Discriminative attention head selection**: Identifies which attention heads are most instance-discriminative
- **Clustering-based layout building**: Constructs countable layout from attention maps
- **Conservative instance adjustment**: Uses heuristic costs (overlap, center, time) to add/remove instances

## Method

Identify misalignments by selecting most instance-discriminative self- and cross-attention heads. Build countable layout via clustering. Use heuristic costs (overlap, center position, temporal consistency) to conservatively add or remove instances. Guide regeneration by modulating cross-attention.

## Results

- **CountBench**: New benchmark for counting accuracy evaluation
- **Wan2.1-1.3B**: +7.4% counting accuracy improvement
- **Wan2.2-5B**: +4.9% improvement
- **Wan2.1-14B**: +5.5% improvement
- Maintains or improves CLIP scores and temporal consistency
