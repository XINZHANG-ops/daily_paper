## Summary

NUMINA addresses a critical limitation in text-to-video diffusion models: the inability to accurately generate the number of objects specified in prompts. While modern T2V models excel at visual fidelity, motion smoothness, and overall semantic alignment, they frequently fail to represent object counts correctly—a problem that hinders their reliability in precision-sensitive applications like instructional visualization. The paper identifies two root causes: semantic weakness where numerical tokens exhibit diffuse cross-attention responses compared to nouns, verbs, and adjectives; and instance ambiguity where heavily downsampled spatiotemporal latent space in DiT architectures limits separability of individual object representations.

The key insight is that discriminative attention heads in T2V models expose critical visual information related to the number of instances, and this can be exploited through a training-free identify-then-guide paradigm. Rather than retraining the model (computationally prohibitive) or relying on external tools, NUMINA operates during generation to identify numerical misalignments via attention analysis and correct them through layout-guided regeneration. The two-phase approach first selects instance-discriminative self-attention heads and text-concentrated cross-attention heads at a reference timestep during pre-generation, then constructs a countable layout by fusing these attention maps and comparing estimated cardinality against prompted numerals. The second phase refines the layout conservatively (adding or removing objects while preserving spatial relationships) and modulates cross-attention through bias adjustments to guide regeneration.

The method is training-free and lightweight, applicable across various T2V models without requiring additional training, datasets, or masks. It achieves up to 7.4% counting accuracy improvement on Wan2.1-1.3B, 4.9% on the 5B model, and 5.5% on the 14B model, while maintaining or improving temporal consistency and CLIP alignment scores.

## Key Contributions

- **Reveals Attention-Based Count Information**: Demonstrates that attention maps in T2V models expose critical visual information related to instance counts, with different attention heads capturing diverse spatial patterns including some that clearly delineate boundaries between instances.

- **Training-Free Identify-Then-Guide Paradigm**: Framework that operates without additional training, masks, or auxiliary networks by exploiting the model's latent ability to separate object instances through attention analysis and explicit layout construction.

- **Instance-Discriminative Attention Head Selection**: Dynamic head selection strategy that selects the self-attention head with highest instance separability and cross-attention head per noun token with most concentrated activation, avoiding the blurred maps produced by naive head averaging.

- **Countable Layout Construction**: Fuses selected attention maps through clustering and overlap-based filtering to create semantic maps where each foreground region corresponds to a single object instance, enabling direct count comparison.

- **Conservative Layout Refinement and Guidance**: Two-step refinement (layout refinement + layout-guided generation) that preserves spatial organization while correcting count errors through attention suppression for removal and bias boosting for addition.

## Method

**Phase 1: Numerical Misalignment Identification**

The method analyzes multi-head attention during early denoising (timestep t* = 20, layer ℓ* = 15) to select discriminative attention heads. For self-attention head selection, each head's map is extracted and PCA-projected onto top three principal components, converted to grayscale. Three complementary scores measure separability: (1) foreground-background separation S1 via standard deviation of intensities; (2) structural richness S2 via variance across partitioned blocks; (3) edge clarity S3 via Sobel operator average gradient magnitude. The overall discriminability score is S(SA_h) = S1 + S2 + γS3, with the head with maximum score selected.

For cross-attention head selection per noun token T, the peak activation (maximum value in the softmax-normalized map) indicates the model's alignment of the token with a specific visual region. The head with highest peak activation is selected, providing computationally efficient and stable identification of relevant regions.

Countable layout construction fuses the selected self-attention map (partitioned into contiguous regions via clustering) and cross-attention map (suppressed below 0.1 peak-ratio threshold, then density-based clustering for focus mask). A region is retained if its semantic overlap score So(ri, F) = |ri ∩ F| / |ri| exceeds threshold τ. The resulting layout MT is a 2D semantic map where each foreground region corresponds to a single object instance, with count |{i: So(ri, F) ≥ τ}|.

**Phase 2: Numerically Aligned Video Generation**

Layout refinement corrects the per-frame layout map MT,f to match target count kT. For object removal, the smallest region is erased as it incurs minimal perturbation. For object addition, a template is selected: if existing regions exist, the smallest is copied as template Ci to preserve scale and shape; otherwise a circle with radius r is used. The template is placed at an optimal location via a cost function balancing overlap (penalizes collisions), center (encourages plausible placement), and temporal terms (ensures stability across frames).

Layout-guided generation modulates cross-attention via softmax(S_pre + B)V where B is a bias term. For removal, B is set to a large negative constant for regions corresponding to removed instances. For addition from circle template, B is set to k·δ(t) where δ is a monotonically decreasing intensity function; for addition from reference region, pre-softmax scores in S_pre are overwritten with the mean attention from the reference.

**Ablation Studies**:
- Reference timestep: t* = 20 provides optimal accuracy-efficiency trade-off; earlier timesteps (10) yield lower accuracy, later timesteps (40+) increase cost without proportional accuracy gains
- Layout construction: Attention-derived layout (49.7%) outperforms GroundingDINO-derived layout (47.5%)
- Layout refinement cost components: Overlap only (45.1%), + Center (46.9%), + Temporal (48.9%), all three (49.7%)
- Head selection: Random (44.1%), All average (43.0%), Top-3 (48.2%), Top-2 (49.4%), Top-1 (49.7%)

**Compatibility with Other Techniques**:
- Combined with seed search and prompt enhancement: 54.2% (+11.9) counting accuracy
- Integrated with EasyCache: reduces wall-clock time from 431s to 355s with 49.4% accuracy

**CogVideoX-5B Evaluation**: NUMINA improves CountAcc from 40.2% to 44.4% (+4.2%), demonstrating generalizability across different DiT implementations.

## Limitations & Future Work

The paper identifies two key limitations. First, NUMINA achieves significant but not perfect accuracy improvement across scenarios—achieving precise numerical alignment for all cases remains challenging. Second, generating very dense instances (tens or hundreds) remains unexplored due to the difficulty of separating individual instances in heavily downsampled latent spaces. The failure case analysis shows that when attention heads focus excessively on the most salient parts of an object (e.g., animal heads) rather than their entirety, over-segmented layouts result where parts of a single instance are mistaken for multiple objects. Future directions include incorporating more holistic perceptual grouping cues, exploring instance-aware attention mechanisms, and extending to higher instance counts.
