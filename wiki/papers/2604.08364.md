# MegaStyle: Constructing Diverse and Scalable Style Dataset via Consistent Text-to-Image Style Mapping

**Published**: 2026-04-08 | **Added**: 2026-04-10 | **arXiv**: [2604.08364](http://arxiv.org/pdf/2604.08364)

## Summary

MegaStyle addresses the problem of缺乏 intra-style consistency, inter-style diversity, and high-quality in existing style datasets. The key insight is leveraging the consistent T2I style mapping capability of modern generative models (specifically Qwen-Image) to generate intra-style consistent, inter-style diverse style pairs.

The pipeline has three stages: (1) Image Pool Collection from LAION-Aesthetics, WikiArt, JourneyDB, (2) Prompt Curation and Balance using Qwen3-VL and hierarchical k-means clustering, (3) Style Image Generation via Qwen-Image. This yields MegaStyle-1.4M with 8,355 artistic styles and 1.4M style images.

## Key Contributions

- **MegaStyle-1.4M dataset**: 8,355 overall artistic styles, 170K style prompts, 400K content prompts, 1.4M style images
- **Style-supervised contrastive learning (SSCL)**: Fine-tunes SigLIP for style-specific representations
- **MegaStyle-Encoder**: Achieves 88.46% mAP@1 on style retrieval, outperforming CLIP (9.29%) and CSD (45.60%)
- **MegaStyle-FLUX**: FLUX-based style transfer with highest text alignment (23.20) and best human preference scores

## Method

Three-stage pipeline: (1) Image pool collection from multiple sources, (2) Prompt curation using Qwen3-VL with balance sampling via hierarchical k-means, (3) Style image generation using Qwen-Image with content-style prompt combinations. SSCL trains MegaStyle-Encoder with combined supervised contrastive loss and image-text contrastive loss.

## Results

- **MegaStyle-Encoder**: 88.46% mAP@1 vs CLIP (9.29%) and CSD (45.60%) on StyleRetrieval benchmark
- **MegaStyle-FLUX**: Highest text alignment (23.20), second-best style alignment (76.16), best human preference scores
- Proves that intra-style consistency, inter-style diversity, and high-quality are all essential for style datasets
