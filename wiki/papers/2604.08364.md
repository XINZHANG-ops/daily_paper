## Summary

MegaStyle presents a novel and scalable data curation pipeline for constructing intra-style consistent, inter-style diverse, and high-quality style datasets for image style transfer. The core innovation lies in leveraging the consistent text-to-image style mapping capability of large generative models like Qwen-Image, which can generate images in the same style from a given style description with high fidelity. This approach overcomes the limitations of previous datasets like IMAGStyle and OmniStyle-150K, which relied on SOTA style transfer methods that produced limited style spaces, inconsistent intra-style pairs, and visible artifacts.

The paper addresses a fundamental challenge in style transfer: existing methods struggle to disentangle style from content in self-supervised training paradigms, leading to content leakage and poor stylized results. The key insight is that paired supervision using high-quality, diverse style pairs that share the same style but differ in content can implicitly model the style transformation more effectively. However, collecting such pairs is difficult because style is multi-dimensional and highly discriminative—minor changes in creation lead to perceptually different styles.

The authors curate a diverse and balanced prompt gallery with 170K style prompts and 400K content prompts, generating up to 68B content-style combinations for training. From this, they construct MegaStyle-1.4M containing 1.4M style images with 8,355 fine-grained artistic styles. The pipeline uses Qwen3-VL for style and content captioning, applies four-level hierarchical k-means clustering with mpnet embeddings for balanced sampling, and employs Qwen-Image for consistent style image generation. The resulting dataset provides intra-style consistency and inter-style diversity that surpasses all previous style datasets.

## Key Contributions

- **Consistent T2I Style Mapping Pipeline**: First exploration of leveraging large generative models' consistent text-to-image style mapping ability to construct style datasets, enabling high-quality intra-style consistent image generation from textual style descriptions.

- **Balanced Prompt Gallery**: 170K style prompts and 400K content prompts curated through VLM-based captioning and four-level hierarchical k-means clustering (50K, 10K, 5K, 1K clusters) using mpnet embeddings, yielding balanced distribution across 8K+ artistic styles.

- **Style-Supervised Contrastive Learning (SSCL)**: Novel training objective to fine-tune MegaStyle-Encoder that combines supervised contrastive learning (SCL) with SigLIP image-text contrastive loss, using batch size 8,192 and temperature 0.07 to extract expressive, style-specific representations.

- **MegaStyle-FLUX**: FLUX-based style transfer model trained with paired supervision on MegaStyle-1.4M, using shifted RoPE on reference style tokens to prevent positional collision and content leakage, achieving state-of-the-art style and text alignment scores.

- **Comprehensive Benchmark Comparison**: Detailed comparison with WikiArt, JourneyDB, Style30K, IMAGStyle, and OmniStyle-150K, demonstrating MegaStyle-1.4M's superior intra-style consistency, inter-style diversity, and quality.

## Method

The data curation pipeline consists of three main stages. First, image pool collection builds content and style image pools from open-source datasets: 1M images from deduplicated JourneyDB spanning diverse Midjourney styles, 80K images from WikiArt covering real-world painting styles, and 1M stylized images from LAION-Aesthetics filtered using WikiArt style descriptors, plus 2M non-stylized images from LAION-Aesthetics for content. Second, prompt curation uses Qwen3-VL-30B-A3B-Instruct to generate captions with specialized instructions for content (objects and visual relationships) and style (color composition, light distribution, artistic medium, texture, brushwork). The balanced sampling employs Exact, Fuzzy, and Semantic deduplication leaving 1M prompts, followed by hierarchical k-means clustering with four levels (50K, 10K, 5K, 1K) and top-down hierarchical sampling for balanced subset formation. Third, style image generation pairs each style prompt with N content prompts using Qwen-Image with FlowMatchScheduler (40 inference steps, cfg=4.0) to synthesize images sharing the same style but containing different content.

MegaStyle-Encoder fine-tuning uses SigLIP-so400m-patch14-384 as the base model, trained for 30 epochs with batch size 8,192, learning rate 5e-4, weight decay 0.01, and temperature 0.07. The SSCL objective combines supervised contrastive loss (Equation 1) with SigLIP image-text contrastive loss (Equation 2), where positive samples share the same style and negatives are sampled from the batch. The training uses large batch size to provide challenging and diverse negative samples.

MegaStyle-FLUX builds on FLUX.1-dev, trained for 30,000 steps with batch size 8, learning rate 1e-4, LoRA rank 128, at 512x512 resolution. The architecture concatenates reference style tokens (VAE-encoded and patchified) with noisy image tokens and text tokens, applying shifted RoPE to reference style tokens to prevent positional collision and cross-image attention bias. Only the diffusion transformer parameters are updated during training.

## Limitations & Future Work

The authors identify several limitations in the current approach. First, when captioning style prompts, VLMs may produce vague words when describing texture, brushwork, and medium elements because the instruction prompt does not specify which visual aspects the VLM should focus on. Second, Qwen-Image shows association bias toward some styles—for example, when the style prompt includes "Japanese painting," generated objects are often depicted as Japanese figures biased toward historical periods such as the Edo or Meiji era. Future work will focus on refining instruction prompts to better cover a broader style space and scaling the style dataset to the 10-million level.
