# WildDet3D: Scaling Promptable 3D Detection in the Wild

**Published**: 2026-04-08 | **Added**: 2026-04-13 | **arXiv**: [2604.08626](http://arxiv.org/pdf/2604.08626)

## Summary

WildDet3D addresses open-vocabulary monocular 3D object detection for general-purpose spatial understanding in real-world environments. The approach uses a unified promptable architecture accepting text/point/box prompts with optional depth input, and curates WildDet3D-Data with 138× more categories than existing benchmarks.

Key innovation is the dual-vision encoders (image + RGBD) with depth fusion module enabling graceful handling of optional depth input at inference time. Also introduces promptable detector supporting text/point/box/exemplar prompts with deeply-supervised 3D detection head.

## Key Contributions

- **Unified promptable architecture**: Text, point, box, and exemplar prompts with optional depth
- **WildDet3D-Data**: 138× more categories than existing benchmarks
- **Dual-vision encoders with depth fusion**: Graceful handling of optional depth input
- **Deeply-supervised 3D detection head**: Unambiguous rotation normalization

## Method

Dual-vision encoders (image + RGBD) with depth fusion module, promptable detector supporting four prompt types, deeply-supervised 3D detection head with unambiguous rotation normalization. WildDet3D-Data built via multi-model candidate generation followed by VLM and human selection.

## Results

- **Omni3D**: 34.2 AP3D (text prompts), 36.4 AP3D (oracle)
- **WildDet3D-Bench**: 22.6 AP3D
- **Argoverse 2/ScanNet zero-shot**: 40.3/48.9 ODS
- **Depth benefit**: +20.7 AP average gain when depth available
- **Training efficiency**: 6-10× fewer epochs than prior methods
