# ClawBench: Can AI Agents Complete Everyday Online Tasks?

**Published**: 2026-04-08 | **Added**: 2026-04-10 | **arXiv**: [2604.08523](http://arxiv.org/pdf/2604.08523)

## Summary

ClawBench evaluates AI agents' ability to perform everyday online tasks (booking flights, shopping, job applications) on live production websites—distinguishing itself from previous benchmarks that use static sandboxes or read-only tasks. The key innovation is a Chrome extension with CDP instrumentation that intercepts final submission requests, enabling safe evaluation without real-world side effects.

The benchmark records five layers of behavior: session replay, screenshots, HTTP traffic, agent messages, browser actions. Evaluation uses an agentic evaluator to compare agent trajectories against human ground truth.

## Key Contributions

- **Real-world evaluation**: Operating on live production websites with write-heavy, state-changing tasks
- **Safe interception**: Chrome extension intercepts final submission requests without real-world side effects
- **Five-layer behavioral recording**: Session replay, screenshots, HTTP traffic, agent messages, browser actions
- **Comprehensive benchmark**: Covers 7 frontier models across diverse everyday online tasks

## Method

Develops Chrome extension with CDP instrumentation to intercept final submission requests. Records five layers of behavior. Uses agentic evaluator to compare trajectories against human ground truth. Safe evaluation without real-world consequences.

## Results

- Even the best model (Claude Sonnet 4.6) achieves only 33.3% success rate
- Large gap between benchmark performance and real-world everyday web task competence
- Previous benchmarks (WebArena, OSWorld) with 65-75% success rates don't reflect real-world performance due to sandbox limitations
