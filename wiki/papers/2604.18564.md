# MultiWorld: Scalable Multi-Agent Multi-View Video World Models

**Published**: 2026-04-19 | **Added**: 2026-04-21 | **arXiv**: [2604.18564](http://arxiv.org/pdf/2604.18564)

## Summary

MultiWorld is a unified framework for multi-agent multi-view world modeling that enables accurate control of multiple agents while maintaining multi-view consistency. The key challenge is that existing video world models assume a single agent in the simulated environment, ignoring interactions and interdependencies among multiple agents. MultiWorld addresses this through the Multi-Agent Condition Module (MACM)—which uses Agent Identity Embedding (via rotary position embeddings) and Adaptive Action Weighting to handle agent distinction—and the Global State Encoder (GSE), which leverages a frozen VGGT backbone to extract 3D-aware global environment states for consistent multi-view generation. The framework scales to arbitrary numbers of agents and camera views without architectural changes.

## Key Contributions

- First scalable multi-agent multi-view video world model framework
- Multi-Agent Condition Module (MACM) for precise multi-agent controllability
- Global State Encoder (GSE) with VGGT backbone for 3D-aware multi-view consistency
- Scalable architecture supporting variable agent and camera counts
- Parallel view generation with ~1.5× speedup over sequential

## Method

**Multi-Agent Condition Module (MACM)**:
- **Agent Identity Embedding (AIE)**: Uses rotary position embeddings to assign distinct identity to each agent's actions; breaks symmetry of multi-agent action space
- **Adaptive Action Weighting (AAW)**: MLP predicts weighting factors for each action token; prioritizes active agents over static ones

**Global State Encoder (GSE)**:
- Uses frozen VGGT (Visual Geometry Grounded Transformer) to encode multi-view observations
- MLP aligns VGGT latents with DiT backbone dimensions
- Compresses variable number of views into compact 3D-aware global representation

**Scalable Framework**:
- Agent scalability: AIE assigns relative identity embeddings, extrapolates to arbitrary agent counts
- View scalability: GSE encodes any number of views into unified global state
- Parallel generation: Different views synthesized simultaneously

## Results

On ItTakesTwo game dataset (100 hours, 100K episodes) and RoboFactory manipulation tasks:

| Metric | MultiWorld | Standard | COMBO |
|--------|-----------|----------|-------|
| FVD (↓) | **179** | 245 | 2070 |
| Action Following (↑) | **89.8%** | 88.4% | 89.3% |
| RPE (↓) | **0.67** | 0.75 | 0.72 |

VGGT backbone in GSE outperforms Wan VAE and DINOv2; base frequency=20 optimal for AIE.

## Limitations & Future Work

- Current scale limited; large-scale training unexplored
- Future: real-time multi-agent generation, ultra-long simulation with memory mechanisms
