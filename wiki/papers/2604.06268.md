# RAGEN-2: Reasoning Collapse in Agentic RL

**Published**: 2026-04-06 | **Added**: 2026-04-09 | **arXiv**: [2604.06268](http://arxiv.org/pdf/2604.06268)

## Summary

This paper identifies and addresses **template collapse** in multi-turn RL training of LLM agents—a failure mode where models produce diverse but input-agnostic reasoning invisible to entropy-based metrics. The key insight is that entropy-based monitoring (the standard approach) cannot detect template collapse because reasoning appears diverse within inputs but becomes input-agnostic across inputs.

The paper proposes **Mutual Information (MI) proxies** to measure cross-input reasoning dependence and explains collapse via a Signal-to-Noise Ratio (SNR) lens. The solution is **SNR-Aware Filtering**, which uses within-prompt reward variance as a lightweight SNR proxy to select high-signal prompts before each policy update.

## Key Contributions

- **Template collapse identification**: A failure mode invisible to entropy-based metrics where reasoning appears diverse within inputs but is input-agnostic across inputs
- **MI-based diagnostics**: Retrieval-Acc and MI-ZScore-EMA proxies for online diagnosis of reasoning quality
- **SNR-Aware Filtering**: Uses within-prompt reward variance to select high-signal prompts for policy updates
- **Superior diagnostics**: MI correlates with performance more strongly than entropy (+0.39 vs. -0.14)

## Method

The core mechanism uses within-prompt reward variance as a proxy for SNR—high variance indicates the prompt generates varied responses worth learning from, while low variance indicates the model has converged to a template response. Before each policy update, only prompts with variance above a threshold are selected for training.

## Results

- Across planning, math reasoning, web navigation, and code execution: MI outperforms entropy as a training signal (+0.39 vs. -0.14 correlation with performance)
- SNR-Aware Filtering consistently improves both input dependence and task success rates across algorithms, model scales, and modalities
- Algorithm-agnostic: works with PPO, DAPO, GRPO and different model sizes

## Personal Notes

This paper's template collapse is closely related to the reader's interest in agent reliability. The key insight is that surface diversity (high entropy) can mask structural collapse—exactly the kind of invisible failure that makes agents unreliable in practice.
