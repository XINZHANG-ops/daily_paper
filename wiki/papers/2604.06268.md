## Summary

This paper identifies and addresses **template collapse**, a failure mode in multi-turn LLM agent reinforcement learning where models produce superficially diverse reasoning that is actually input-agnostic. The key insight is that entropy—widely used to monitor reasoning stability—only measures diversity within the same input and cannot tell whether reasoning actually responds to different inputs. Even with stable entropy, models can rely on fixed templates that look diverse within any single input but are effectively the same across inputs.

The paper decomposes reasoning quality into two axes: **within-input diversity** (measured by conditional entropy $H(Z|X)$) and **cross-input distinguishability** (measured by mutual information $I(X; Z)$). Template collapse manifests as high conditional entropy but low mutual information—reasoning appears diverse within each input but becomes input-agnostic across inputs. This is invisible to all existing entropy-based metrics.

The authors explain template collapse through a **signal-to-noise ratio (SNR)** mechanism: low reward variance weakens task gradients while input-agnostic regularization (KL divergence and entropy regularization) remains constant, causing regularization to dominate updates and erase cross-input reasoning differences. To address this, they propose **SNR-Aware Filtering**, which uses reward variance as a lightweight SNR proxy to select high-signal prompts each iteration.

## Key Contributions

1. **Identification of template collapse**: First formal characterization of a failure mode where reasoning appears diverse within inputs but becomes input-agnostic across inputs, invisible to entropy-based metrics.

2. **Mutual Information proxy family**: A family of diagnostic metrics (Retrieval-Acc, MI-Est, MI-ZScore-EMA) that measure cross-input distinguishability without external models, computed via in-batch cross-scoring of reasoning traces against prompts.

3. **SNR mechanism explanation**: Formal analysis showing that low reward variance weakens task gradients while regularization gradients remain flat, causing input-agnostic patterns to dominate.

4. **SNR-Aware Filtering**: Mitigation method that filters prompts by reward variance before each update, concentrating gradient budget on high-SNR prompts.

5. **Comprehensive validation**: Across planning (Sokoban), math reasoning (MetaMathQA, Countdown), web navigation (WebShop), and code execution (DeepCoder), under multiple RL algorithms (PPO, DAPO, GRPO, Dr. GRPO), model scales (0.5B to 7B), and modalities (text and image inputs).

## Method

### Problem Setup

The paper studies closed-loop multi-turn agent RL where a policy $\pi_\theta$ is trained by rolling out trajectories and updating on collected experience. At each time step $t$, the agent observes $o_t$, generates reasoning tokens $z_t$ and action $a_t$, and receives reward $r_t$, forming trajectory $\tau = \{(o_t, z_t, a_t, r_t)\}_t$.

The standard PPO/GRPO objective contains regularization terms (KL divergence, entropy bonus) that act uniformly across all inputs regardless of content:

$$\mathcal{L}(\theta) = \mathbb{E}_{x, \tau}[A(\tau, x) - \lambda_{KL} D_{KL}(\pi_\theta \parallel \pi_{ref}) + \lambda_H H(\pi_\theta)]$$

### Mutual Information Proxy Family

The key insight is that marginal entropy $H(Z)$ decomposes via:
$$H(Z) = I(X; Z) + H(Z|X)$$

Where $I(X; Z)$ is input dependence (mutual information between input $X$ and reasoning $Z$), and $H(Z|X)$ is within-input diversity (conditional entropy given input). Entropy metrics only proxy $H(Z|X)$, missing declines in $I(X; Z)$.

**In-Batch Cross-Scoring**: Given $P$ prompts and $G$ reasoning samples per prompt, compute teacher-forced log-likelihoods for every $(Z_{i,k}, X_j)$ pair, forming scoring matrix $L_{i,k,j} = \log p_\theta(Z_{i,k} | X_j)$. Extract:
- **matched**: per-token log-likelihood under true source input
- **marginal**: per-token log-likelihood under uniform prompt mixture

**Retrieval-Acc**: Fraction of reasoning traces correctly matched to their source prompt. Under template collapse, approaches chance level $1/P$ (1.56% at $P=64$).

**MI-ZScore-EMA**: Input dependence estimated as z-score normalized and EMA-smoothed difference between matched and marginal probabilities:
$$\hat{I}(X;Z) = \frac{1}{PG}\sum_{i,k} \frac{matched_{i,k} - marginal_{i,k}}{\sigma_{batch}+\epsilon}$$

### SNR Mechanism

The gradient analysis shows three patterns consistently across algorithms:

1. **Task gradient scales with reward variance**: $\|g_{task}\|$ increases monotonically with bucket RV. High-variance prompts yield strong task-discriminative gradients; low-variance prompts produce weak gradients even when non-zero.

2. **Regularization gradient is flat**: $\|g_{reg}\|$ (from KL and entropy terms) remains constant across all buckets, applying uniform contraction to every reasoning chain regardless of source prompt.

3. **Low-RV prompts produce regularization-dominated updates**: In lowest-variance buckets, task gradients nearly vanish while regularization gradients persist, so updates are driven entirely by input-agnostic noise.

Formal bound via Cauchy-Schwarz: $|g_{task}(x)| \leq \sqrt{dVar(R|X=x)} \cdot C$. Low reward variance weakens $g_{task}$ while leaving $g_{reg}$ unchanged, driving $I(X; Z) \rightarrow 0$.

### SNR-Aware Filtering

At each iteration, estimated $Var(R|X)$ for each prompt and retain only the top fraction by variance before computing parameter updates:

$$S = \{\sigma(1), ..., \sigma(k^*)\}, \text{ where } k^* = \min\{k: \sum_{j=1}^{k} dVar(R|X=x_{\sigma(j)}) \geq \tau\}$$

The threshold $\tau = \rho \sum_{i=1}^{P} dVar(R|X=x_i)$ ensures adaptive selection based on variance mass rather than fixed count.

## Results

### Template Collapse as a Consistent Failure Mode

Across all training configurations, RL-trained agents reliably develop reasoning that is fluent but input-agnostic: $I(X; Z)$ declines while $H(Z|X)$ remains high, invisible to entropy-based monitoring. The MI proxy serves as an early warning signal, detecting reasoning degradation before task performance degrades.

### Diagnostic Metric Correlations

Spearman correlation with task performance:
- **Trajectory MI-ZScore**: +0.39 (positive, reliable)
- **Reasoning Entropy**: -0.11 (misleading)
- **Conditional Entropy**: -0.14 (misleading)

MI-family metrics predict performance twice as reliably as entropy, and entropy actually points in the wrong direction.

### Quartile Ablation (Causal Evidence)

Training on prompts sorted by RV into quartiles shows monotonic degradation from Q1 to Q4:

## Limitations & Future Work

1. **RV requires G≥2 trajectories per prompt**: SNR-Aware Filtering needs sufficient trajectories per prompt to estimate within-prompt reward variance, limiting configurations with G=1.

2. **When RV is uniformly low**: In environments with very high stochasticity (80-100%), RV loses discriminative power as even high-effort prompts yield noisy rewards.

3. **Format validity vs content sensitivity**: Structural correctness (format validity) and semantic input-dependence are separate dimensions—high format validity does not guarantee high MI.

4. **Orthogonal to entropy/KL tuning**: SNR-Aware Filtering acts as a signal-enhancement mechanism, complementing noise-control mechanisms like KL and entropy regularization, which move training dynamics along different axes.
