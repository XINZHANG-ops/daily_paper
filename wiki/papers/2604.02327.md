## Summary

SteerViT introduces a paradigm shift for visual representation learning: instead of conditioning language models on vision (as in MLLMs) or fusing visual and text features after encoding (as in CLIP), SteerViT conditions pretrained Vision Transformers on language through early fusion via lightweight cross-attention layers interleaved within frozen ViT blocks. This produces a new class of visual representations—"Steerable Visual Representations"—whose global and local features can be steered with natural language while preserving the underlying representation quality of the frozen backbone.

The method addresses a fundamental limitation of standard ViTs: they tend to focus on the most salient objects in an image (photographer bias) with no mechanism to direct attention elsewhere. While MLLMs can be steered with text, their representations are language-centric and lose effectiveness for generic visual tasks. SteerViT achieves a new Pareto frontier: 96% retrieval accuracy on CORE (vs 44% for vanilla DINOv2) while preserving representation quality that matches or exceeds the frozen backbone on downstream tasks.

The architecture adds only ~21M trainable parameters (two orders of magnitude fewer than billion-parameter MLLMs) via gated cross-attention layers that attend from visual patch tokens to text token embeddings. Training uses referential image segmentation as a pretext task to encourage vision-language alignment, with 162k unique images and 2.28M image-text pairs from RefCOCO/+/g, Visual Genome, LVIS, and Mapillary Vistas.

## Key Contributions

- **Early Vision-Language Fusion**: Conditions frozen ViT internal features on text via interleaved gated cross-attention layers, rather than late fusion after encoding—text influences visual processing even at early layers
- **Pareto Improvement Over Prior Work**: Achieves both high steerability (96% on CORE) and preserved representation quality (87.7% FG-CLS vs 89.0% vanilla DINOv2)—the first model to satisfy all three desiderata: steerability, representation quality, and early V-L fusion
- **Parameter Efficiency**: Adds only 21M trainable parameters vs 1B+ for MLLMs while outperforming them on steerability; enables zero-shot transfer to new domains without task-specific training
- **Zero-Shot Generalization**: Matches or surpasses dedicated methods on personalized object discrimination (58.1% vs 48.0% PR-AUC for fine-tuned DINOv2), anomaly detection (82.1% PRO on MVTec AD vs 84.5% for specialist FADE), and text-guided retrieval
- **Continuous Control via Gating**: Tanh-gated cross-attention with zero-initialized gates allows smooth interpolation between vanilla ViT and fully text-conditioned state via scaling factor ω ∈ [0,1]

## Method

**Architecture**:
- **Visual Encoder**: Frozen pretrained ViT (DINOv2 ViT-B/14 default; also supports SigLIP, MAE) produces patch tokens Z_v ∈ ℝ^(N×d_v)
- **Text Encoder**: Frozen pretrained RoBERTa-Large produces token-level embeddings Z_t ∈ ℝ^(L×d_t) for conditioning prompts
- **Multimodal Adapter**: 2-layer MLP projects L×d_t text embeddings to L×d_v visual-aligned space H_t
- **Gated Cross-Attention**: Inverted Flamingo formulation where visual patch tokens attend to language tokens; inserted into every other ViT transformer block. Output integrated via tanh gate with learnable α_ℓ initialized to zero (ensuring identity at initialization)

**Formulation**:
```
Ẑ_v^(ℓ) = CA(Z_v^(ℓ), H_t) = softmax(QK^⊤ / √d_k) V
where Q = Z_v^(ℓ) W_Q, K = H_t W_K, V = H_t W_V

Z_v^(ℓ+1) = Z_v^(ℓ) + tanh(α_ℓ) · Ẑ_v^(ℓ)
```

**Training Objective**: Referential segmentation—given image and text prompt referring to a target, predict which patches correspond to the referred region. Ground truth is patchified segmentation mask; soft cross-entropy loss encourages routing of textual information into visual patch tokens.

**Training Data**: 162k unique images, 2.28M pairs from RefCOCO/+/g, Visual Genome, LVIS, Mapillary Vistas.

## Results

**CORE Benchmark** (COnditional REtrieval, one-vs-all retrieval of non-salient objects):
| Method | Acc@1 |
|--------|-------|
| SteerViT | **96.0%** |
| SAM3 | 93.3% |
| FLAIR | 81.3% |
| DINOv2 | 44.0% |
| CLIP | 44.2% |
| Random | 20.0% |

**Representation Quality vs Steerability Trade-off**:
| Method | Steerability | Representation Quality |
|--------|-------------|------------------------|
| SteerViT | 96.0% | 87.7% |
| DINOv2 (vanilla) | 43.7% | 89.0% |
| CLIP | 44.2% | ~80% |
| MLLMs | 47-76% | 60-80% |

**Downstream Task Performance**:
- **Personalized Object Discrimination** (PODS): 58.1% PR-AUC with descriptive prompts vs 48.0% for task-specific fine-tuned DINOv2
- **MVTec AD Anomaly Segmentation**: 82.1% PRO, matching dedicated methods (FADE: 84.5%)
- **MOSAIC Attention Steering**: 50.2% PR-AUC vs 14.3% for DINOv2 when steering attention to non-salient objects

**Architecture Ablations**:
- Early fusion > late fusion (+2.7 CORE points, +21.5 PODS points)
- Tanh gating essential (+1.4 CORE, +11.0 PODS vs ungated)
- 2-layer MLP projector > linear (+1.0 CORE, +1.7 PODS)

## Limitations & Future Work

- **Backbone Dependence**: SteerViT conclusions are coupled to the capabilities of frozen ViT backbones; stronger recent-context performance may partly reflect DINOv2's rich self-supervised features
- **Training Compute**: 500k iterations (~84 H100 GPU-hours) for DINOv2-ViT-B/14; scaling to larger ViTs increases training cost
- **Text Encoder Choice**: Currently fixed to RoBERTa-Large; exploring lighter text encoders could reduce the 21M parameter overhead
- **Scope as Visual Representation Framework**: Not a proposal for new memory mechanisms or retrieval designs; focuses on extending frozen vision encoders with language-conditioned steering
- **Generalization Beyond ViT**: Experiments focus on transformer-based vision encoders; extension to CNN-based architectures unexplored
