# RationalRewards: Reasoning Rewards Scale Visual Generation Both Training and Test Time

**Published**: 2026-04-12 | **Added**: 2026-04-17 | **arXiv**: [2604.11626](http://arxiv.org/pdf/2604.11626)

## Summary

Most visual generation reward models reduce rich human judgments to a single unexplained score, discarding the structured reasoning that underlies preference. RationalRewards proposes a fundamental shift: teaching reward models to produce explicit, multi-dimensional critiques before scoring, transforming them from passive evaluators into active optimization tools.

The key innovation is Preference-Anchored Rationalization (PARROT), a variational framework that trains reasoning-based reward models from readily available preference data without costly rationale annotations. The 8B-parameter model achieves state-of-the-art preference prediction competitive with Gemini-2.5-Pro while using 10-20× less training data than comparable baselines.

## Key Contributions

- **PARROT framework**: Three-phase pipeline (rationale generation, consistency filtering, distillation) that treats rationales as latent variables recoverable from preference data
- **Dual-space optimization**: Enables both parameter-space RL and test-time prompt refinement
- **Reward hacking resistance**: Structured multi-dimensional reasoning prevents the single-score exploitation that plagues scalar reward models
- **Test-time scaling insight**: Generate-Critique-Refine loop can match or exceed RL fine-tuning, suggesting generators have latent capabilities under-elicited by suboptimal prompts

## Method

**PARROT Training Pipeline:**
1. **Rationale Generation (Hindsight)**: Teacher VLM (Qwen3-VL-32B) generates candidate rationales anchored to known preference labels
2. **Consistency Filtering**: Retain only rationales where preference can be recovered from (x, z) alone—filters hallucinations
3. **Student SFT (Foresight)**: Train 8B student to generate rationales without seeing the answer

**Dual-Space Optimization:**
- **Parameter Space**: Multi-dimensional scores provide dense RL rewards via DiffusionNFT
- **Prompt Space**: Generate-Critique-Refine loop refines prompts iteratively based on RationalRewards' critique

## Results

| Setting | Result |
|---------|--------|
| Preference prediction | SOTA among open-source (competitive with Gemini-2.5-Pro) |
| Training data | 10-20× less than comparable baselines |
| RL fine-tuning | +9.37 points on UniGenBench++ (FLUX.1-dev: 60.97 → 70.34) |
| Test-time prompt tuning | Matches or exceeds RL fine-tuning without parameter updates |

**Key insight**: The latent capability hypothesis—generators possess capabilities under-elicited by suboptimal prompts. Structured critiques bridge user intent and model capability without weight modification.

## Limitations & Future Work

- Teacher model quality upper-bounds student quality
- Preference datasets encode annotator biases
- Latent capability hypothesis needs representation-level validation
- Threshold sensitivity (3.0) may vary by task
