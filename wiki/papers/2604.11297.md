# The Past Is Not Past: Memory-Enhanced Dynamic Reward Shaping

**Published**: 2026-04-12 | **Added**: 2026-04-16 | **arXiv**: [2604.11297](http://arxiv.org/pdf/2604.11297)

## Summary

MEDS addresses reduced sampling diversity in on-policy RL optimization, where the policy collapses into a narrow "stable error basin" and repeatedly generates similar erroneous behaviors. The solution captures historical error patterns via layer-wise logits clustering and dynamically penalizes recurrent failure modes.

The key insight is using HDBSCAN clustering on layer-wise logits of the final answer token to identify and penalize similar error patterns, encouraging broader exploration.

## Key Contributions

- **Memory-enhanced framework**: Captures historical error patterns via layer-wise logits clustering
- **HDBSCAN clustering**: Identifies similar error patterns for dynamic penalty
- **Stable error basin detection**: Identifies when policy collapses into repeating errors
- **Strong results across model scales**: Works on Qwen3-1.7B, Qwen2.5-Math-7B, and Qwen3-8B

## Method

Extracts logic feature vectors from layer-wise logits of the final answer token. Maintains per-prompt error memory updated online. Applies HDBSCAN clustering to group similar error patterns. Adjusts rewards by subtracting penalties proportional to cluster size to encourage broader exploration.

## Results

Across three models (Qwen3-1.7B, Qwen2.5-Math-7B, Qwen3-8B) on five benchmarks:
- Up to 4.13 pass@1 improvement
- Up to 4.37 pass@128 improvement
- Increased behavioral diversity confirmed
- Strong correlation between clustering quality and downstream performance
