# Pseudo-Unification: Entropy Probing Reveals Divergent Information Patterns in Unified Multimodal Models

**Published**: 2026-04-12 | **Added**: 2026-04-14 | **arXiv**: [2604.10949](http://arxiv.org/pdf/2604.10949)

## Summary

This paper investigates a fundamental limitation in Unified Multimodal Models (UMMs) — models designed to synergize LLMs with vision models for both text generation and image synthesis. The authors discover that despite sharing parameters, UMMs exhibit "pseudo-unification": text generation shows high-entropy creativity while image synthesis enforces low-entropy fidelity. Using a novel information-theoretic probing framework based on matrix-based Rényi entropy in RKHS, they reveal this stems from dual divergence: modality-asymmetric encoding (vision and language follow divergent entropy trajectories) and pattern-split response (text vs image generation follow different response patterns). Only models like Harmon (using contextual prediction for both modalities) achieve genuine unification, demonstrating that real multimodal synergy requires consistent information flow, not just shared parameters.

## Key Contributions

- First model-internal probing framework for diagnosing unification in UMMs using information-theoretic measures
- Matrix-based Rényi entropy reformulation enabling entropy computation in implicit high-dimensional spaces without density estimation
- Discovery that pseudo-unification stems from dual divergence: modality-asymmetric encoding and pattern-split responses
- Demonstration that only models unifying both encoding and generative logic (e.g., via contextual prediction) achieve genuine unification

## Method

The paper proposes a two-level entropy-probing framework:

1. **Prompt-Level Analysis**: Measures embedding entropy and layer-wise entropy to reveal modality-specific encoding patterns. Uses matrix-based Rényi entropy in Reproducing Kernel Hilbert Spaces (RKHS) to compute entropy from kernel matrices without requiring explicit probability densities.

2. **Response-Level Analysis**: Computes conditional entropy proxy (Ĥ(Zr|Zp) = Hα(Kjoint) - Hα(Kpp)) to diagnose response behavior divergence. Lower values indicate fidelity-driven generation (image synthesis), higher values indicate creative responses (text generation).

The key insight is that Transformer-based UMMs don't expose explicit joint densities, so classical entropy estimation is infeasible. The authors bridge this gap by treating embedding sequences as empirical samples, modeling similarity via Gaussian kernels, and reinterpret entropy as a geometric property of representation structure rather than a function of probability density.

## Results

Analyzed ten representative UMMs including BAGEL (14B), Harmon (1.5B), Janus-Pro (7B/1B), JanusFlow (1.3B), Show-o (1.3B), Show-o2 (7B), and OmniGen2 (7B).

Key findings:
- Pseudo-unification is widespread: most UMMs show divergent entropy trajectories between text and image modalities
- Architecture matters: all-in-one transformers with contextual prediction (Harmon) achieve genuine unification; MoT-based models (BAGEL) and diffusion-pretrained models show significant divergence
- Smaller models can outperform: Harmon (1.5B) outperforms BAGEL (14B) in cross-modal reasoning due to consistent information flow
- Text generation: high-entropy, creative, exploring many possibilities
- Image synthesis: low-entropy, fidelity-driven, focused on prompt alignment

## Limitations & Future Work

The paper acknowledges that their entropy-based approach is a proxy for true Shannon entropy, though empirically validated. Future work could explore:
- Architectures that explicitly unify encoding and response patterns
- Training objectives that encourage consistent information flow across modalities
- Extension to video generation and other modalities beyond text and images
