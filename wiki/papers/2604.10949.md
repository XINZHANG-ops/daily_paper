# Pseudo-Unification: Entropy Probing Reveals Divergent Information Patterns in Unified Multimodal Models

**Published**: 2026-04-12 | **Added**: 2026-04-14 | **arXiv**: [2604.10949](http://arxiv.org/pdf/2604.10949)

## Summary

This paper investigates **pseudo-unification** in Unified Multimodal Models (UMMs)—despite shared parameters, models exhibit divergent response behaviors where text generation shows high-entropy creativity while image synthesis enforces low-entropy fidelity. The root causes are **modality-asymmetric encoding** and **pattern-split responses**.

The paper proposes an information-theoretic framework using matrix-based Rényi entropy in Reproducing Kernel Hilbert Spaces (RKHS) to jointly analyze how UMMS encode inputs and generate outputs across modalities. The key finding is that only **Harmon** (using contextual prediction for both modalities) achieves genuine unification.

## Key Contributions

- **Pseudo-unification identification**: Despite shared parameters, text and image generation show divergent behaviors
- **Information-theoretic framework**: Matrix-based Rényi entropy in RKHS for analyzing multimodal encoding
- **Modality-asymmetric encoding**: Root cause of pseudo-unification in UMMS
- **Harmon as exception**: Only model achieving genuine unification via contextual prediction for both modalities

## Method

Two-level entropy-probing framework: (i) Prompt-level analysis measuring embedding entropy and layer-wise entropy to reveal modality-specific encoding patterns; (ii) Response-level analysis computing conditional entropy proxy to diagnose response behavior divergence. Analyzed across 10 representative UMMS.

## Results

- Across 10 UMMS, pseudo-unification arises from dual divergence in encoding and response patterns
- Only Harmon achieves genuine unification through contextual prediction for both vision and language
- Real multimodal synergy requires consistent information flow, not just shared parameters
