# SpatialEvo: Self-Evolving Spatial Intelligence via Deterministic Geometric Environments

**Published**: 2026-04-14 | **Added**: 2026-04-16 | **arXiv**: [2604.14144](http://arxiv.org/pdf/2604.14144)

## Summary

SpatialEvo addresses 3D spatial reasoning in VLMs for embodied intelligence. The key innovation is the **Deterministic Geometric Environment (DGE)** that exploits the property that spatial ground truth is a deterministic consequence of geometry computable from point clouds and camera poses—replacing model consensus with objective physical feedback.

The problem with existing self-evolving methods is that they rely on model voting to construct pseudo-labels, which causes training to reinforce rather than correct the model's own geometric errors. DGE provides exact ground truth without model involvement.

## Key Contributions

- **Deterministic Geometric Environment (DGE)**: Computes exact ground truth from 3D geometry without model involvement
- **Zero-noise interactive oracles**: 16 spatial reasoning task categories with geometric validation rules
- **Single policy model co-evolution**: Questioner and Solver roles under GRPO with adaptive task scheduling
- **Pure online RL**: No supervised fine-tuning phase

## Method

DGE transforms 3D scenes into zero-noise interactive oracles using 16 spatial reasoning task categories and geometric validation rules. Single policy model co-evolves as Questioner and Solver under GRPO with adaptive task scheduling. Trained entirely via online reinforcement learning without supervised fine-tuning.

## Results

- **3B scale**: 51.1 average score across nine benchmarks
- **7B scale**: 54.7 average score, highest across benchmarks
- **VSI-Bench**: 46.1 (7B scale)
- **Ablation confirms**: DGE physical grounding is the largest contributor to performance
