# ClawGUI: A Unified Framework for Training, Evaluating, and Deploying GUI Agents

**Published**: 2026-04-12 | **Added**: 2026-04-15 | **arXiv**: [2604.11784](http://arxiv.org/pdf/2604.11784)

## Summary

ClawGUI addresses three critical gaps in GUI agent research: (1) lack of open-source RL training infrastructure for virtual and physical devices, (2) poorly aligned evaluation protocols causing irreproducible results, and (3) absence of deployment pathways to real users on real devices. The unified framework provides scalable RL training (ClawGUI-RL), standardized evaluation (ClawGUI-Eval), and production deployment (ClawGUI-Agent).

Key innovation is **GiGPO** advantage estimation with **Process Reward Model** for dense step-level supervision across parallel virtual and real device environments.

## Key Contributions

- **ClawGUI-RL**: Scalable online RL with GiGPO advantage estimation and Process Reward Model for dense step-level supervision
- **ClawGUI-Eval**: Standardized three-stage Infer→Judge→Metric pipeline across 6 benchmarks
- **ClawGUI-Agent**: Hybrid CLI-GUI control through 12+ chat platforms with persistent personalized memory
- **Open-source unified framework**: End-to-end solution from training to evaluation to deployment

## Method

Three-module architecture: (1) ClawGUI-RL provides online RL with GiGPO advantage estimation across parallel virtual and real device environments, with Process Reward Model for dense step-level supervision; (2) ClawGUI-Eval enforces standardized three-stage Infer→Judge→Metric pipeline; (3) ClawGUI-Agent enables deployment via hybrid CLI-GUI control.

## Results

- **ClawGUI-2B**: 17.1% Success Rate on MobileWorld GUI-Only, +6.0 absolute points over MAI-UI-2B
- **GiGPO**: 17.9% relative improvement over episode-level GRPO
- **ClawGUI-Eval**: 95.8% reproduction rate against official baselines
