MergeVQ: 视觉生成与表示的统一框架

MergeVQ: A Unified Framework for Visual Generation and Representation with Disentangled Token Merging and Quantization

发布日期: 2025-04-01

链接: http://arxiv.org/pdf/2504.00999

1. 📘 主题与领域: 一种称为MergeVQ的统一框架，用于视觉生成和表示学习，结合了令牌合并技术和向量量化在计算机视觉中的应用。
2. 💡 以往研究与新思路: 基于向量量化（VQ）和掩码图像建模（MIM）研究，提出了解耦合的令牌合并和量化的新思路，弥合了生成和表示学习任务之间的差距。
3. ❓ 问题: 解决共享潜在空间中生成质量和表示学习能力之间的权衡，同时提高两种任务的效率。
4. 🛠️ 方法: 使用带有无查找表量化（LFQ）的令牌合并进行压缩，引入源恢复以保留空间信息，并采用具有KV缓存压缩的MergeAR进行高效生成。
5. 📊 结果与评估: 在ImageNet-1K上实现了表示学习（79.8%线性探测准确率）和图像生成（gFID为2.24）的竞争性能，同时保持了有利的令牌效率和推理速度。
